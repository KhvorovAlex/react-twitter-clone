Копия популярной социальной сети Twitter-clone - [Demo](https://react-twitter-clone-gules.vercel.app/)

В проекте использован следующий стек технологий (по мере выполнения, будет дополняться):

**Frontend:**

-   TypeScript
-   ReactJS
-   Material UI
-   React Router
-   classnames
-   redux
-   redux-saga
-   reselect
-   immer
-   axios

**Hooks:**

-   useState
-   useEffect
